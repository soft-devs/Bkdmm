# ER 图功能设计文档

## 一、工具栏按钮

### 1. 模式切换区
| 按钮 | 图标 | 功能 | 说明 |
|------|------|------|------|
| 移动模式 | `Icons.pan_tool` (手掌) | 切换到移动/预览模式 | 激活时显示 primary 主题色 |
| 编辑模式 | `TDIcons.edit` | 切换到编辑模式 | 激活时显示 primary 主题色 |

### 2. 缩放控制区
| 按钮 | 图标 | 功能 | 说明 |
|------|------|------|------|
| 放大 | `TDIcons.zoom_in` | 放大画布 | 每次放大 20% |
| 缩小 | `TDIcons.zoom_out` | 缩小画布 | 每次缩小 20% |
| 适应屏幕 | `TDIcons.fullscreen` | 自动缩放以适应所有节点 | 调用 `zoomToFit()` |

### 3. 布局控制区
| 按钮 | 图标 | 功能 | 说明 |
|------|------|------|------|
| 自动布局 | `TDIcons.view_module` | 自动排列节点 | 使用 Sugiyama 层次布局算法 |

## 二、预览模式（移动模式）功能

### 核心原则
**只可查看，不可编辑**

### 功能列表

| 功能 | 操作方式 | 说明 |
|------|----------|------|
| 平移画布 | 鼠标拖拽空白区域 | 移动整个画布视图 |
| 缩放画布 | 鼠标滚轮 / 工具栏按钮 | 放大或缩小视图 |
| 查看节点 | 无操作 | 显示所有表节点及其字段 |
| 查看关系线 | 无操作 | 显示实体之间的关系连线 |
| 查看关系标记 | 无操作 | 显示 1:N、N:M 等关系类型标记 |
| 鼠标坐标显示 | 鼠标移动 | 左下角实时显示当前鼠标坐标 |
| 选择节点 | 单击节点 | 高亮选中节点（仅视觉反馈，不触发编辑） |
| 打开实体编辑 | 双击节点 | 打开实体编辑弹窗（允许查看和修改实体字段） |

### 禁止操作
- ❌ 拖动节点位置
- ❌ 显示字段锚点
- ❌ 创建新关系连线
- ❌ 删除节点或关系

---

## 三、编辑模式功能

### 核心原则
**可编辑节点位置和关系连线**

### 功能列表

| 功能 | 操作方式 | 说明 |
|------|----------|------|
| 拖动节点 | 鼠标拖拽节点主体 | 移动节点位置，自动保存 |
| 显示锚点 | 自动显示（编辑模式下） | 每个字段左右两侧显示连接锚点 |
| 开始连线 | 点击字段锚点 | 选中源锚点，开始连线预览 |
| 连线预览 | 鼠标移动 | 从源锚点到鼠标位置显示虚线预览 |
| 完成连线 | 点击目标锚点 | 创建关系，弹出关系类型选择对话框 |
| 取消连线 | 点击空白区域 / 按 ESC | 取消当前连线操作 |
| 选择节点 | 单击节点 | 高亮选中节点 |
| 打开实体编辑 | 双击节点 | 打开实体编辑弹窗 |
| 平移画布 | 鼠标拖拽空白区域 | 移动整个画布视图（与节点拖动区域区分） |
| 缩放画布 | 鼠标滚轮 / 工具栏按钮 | 放大或缩小视图 |
| 自动布局 | 工具栏按钮 | 自动排列所有节点位置 |

### 锚点设计

| 属性 | 说明 |
|------|------|
| 位置 | 字段行左右两侧，超出节点边界 8px |
| 视觉大小 | 6px 圆形 |
| 点击区域 | 20px（增加点击容错范围） |
| 颜色 | 主键字段：金色；普通字段：蓝色 |
| 悬停光标 | `SystemMouseCursors.cell` (十字准星) |

### 关系类型选择对话框

| 字段 | 说明 |
|------|------|
| 源字段 | 显示源表名和字段名 |
| 目标字段 | 显示目标表名和字段名 |
| 关系类型 | 默认 1:N，未来可选择 1:1、N:M |
| 标签 | 可选，为关系添加备注标签 |

## 四、节点（表）显示内容

### 表头区域
| 内容 | 说明 |
|------|------|
| 表图标 | `Icons.table_rows` (表格行图标) |
| 表名 | 实体的 `title` 字段（英文表名） |
| 中文名 | 实体的 `chnname` 字段（可选显示） |

### 字段行区域
| 内容 | 说明 |
|------|------|
| 主键图标 | `Icons.vpn_key` (钥匙图标)，仅主键字段显示 |
| 字段名 | `field.name` |
| 字段类型 | 格式化为 `type(length, decimal)` |

### 视觉样式
| 属性 | 值 |
|------|------|
| 默认宽度 | 200px |
| 表头高度 | 40px |
| 字段行高度 | 28px |
| 圆角半径 | 8px |
| 阴影 | 黑色 15% 透明度，偏移 (2, 2)，模糊 6px |
| 选中边框 | 主题色，宽度 2px |

## 五、关系线（边）显示

### 连线样式
| 属性 | 值 |
|------|------|
| 线条类型 | 直线或折线（根据锚点位置） |
| 线条宽度 | 2px |
| 线条颜色 | 暗色模式：灰色 600；亮色模式：灰色 |

### 关系标记
| 类型 | 源端标记 | 目标端标记 |
|------|----------|------------|
| 1:1 | "1" | "1" |
| 1:N | "1" | "N" |
| N:1 | "N" | "1" |
| N:M | "N" | "M" |

### 鸦脚标记（Crow's Foot）
- N/M 端显示鸦脚符号（三条分叉线）
- 1 端显示单横线

## 七、交互流程图

### 预览模式交互
```
用户进入 ER 图
    │
    ▼
默认预览模式
    │
    ├── 拖拽空白 → 平移画布
    ├── 滚轮 → 缩放画布
    ├── 单击节点 → 高亮选中
    ├── 双击节点 → 打开实体编辑弹窗
    └── 工具栏按钮 → 缩放/布局
```

### 编辑模式交互
```
用户点击编辑按钮
    │
    ▼
切换到编辑模式
    │
    ├── 显示所有字段锚点
    │
    ├── 拖拽节点 → 移动节点位置
    │   │
    │   └── 释放 → 保存位置到项目
    │
    ├── 点击锚点 → 开始连线
    │   │
    │   ├── 移动鼠标 → 显示连线预览
    │   │
    │   ├── 点击目标锚点 → 创建关系
    │   │   │
    │   │   └── 弹出关系类型对话框
    │   │       │
    │   │       └── 确认 → 保存关系
    │   │
    │   └── 点击空白/ESC → 取消连线
    │
    ├── 双击节点 → 打开实体编辑弹窗
    │
    └── 工具栏按钮 → 缩放/布局
```

## 九、状态指示

### 模式指示
- 工具栏按钮高亮显示当前模式
- 编辑模式下锚点可见

### 连线状态指示
- 连线过程中：鼠标位置显示虚线预览
- 源锚点高亮

### 拖动状态指示
- 拖动时：节点跟随鼠标移动
- 悬停光标：抓手 (`grab`) / 抓取中 (`grabbing`)

### 选择状态指示
- 选中节点：边框高亮（主题色，2px）
- 多选节点：所有选中节点边框高亮

## 十、数据持久化

### 自动保存时机
| 操作 | 保存内容 |
|------|----------|
| 移动节点位置 | GraphNode.x, GraphNode.y |
| 创建关系连线 | GraphEdge (source, target, relationType) |
| 自动布局 | 所有节点的位置 |

### 保存路径
```
Project
  └── modules[index]
        └── graphCanvas
              ├── nodes[i].x, y  (节点位置)
              └── edges[i]       (关系连线)
```
