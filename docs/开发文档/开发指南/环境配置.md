# 环境配置

> 阅读时机: 项目初始化、配置开发环境

---

## Flutter 安装

```bash
# Windows: 下载 SDK，添加 PATH
# macOS: brew install flutter
# Linux: snap install flutter --classic

flutter doctor
```

---

## 项目初始化

```bash
flutter create bkdmm --platforms=windows,macos,linux
cd bkdmm
flutter pub get
```

---

## 依赖配置

```yaml
dependencies:
  flutter_riverpod: ^2.4.0
  hive_flutter: ^1.1.0
  file_picker: ^6.1.0
  syncfusion_flutter_datagrid: ^24.1.0
  mustache_template: ^2.0.0
  uuid: ^4.4.0

dev_dependencies:
  flutter_lints: ^3.0.0
  build_runner: ^2.4.0
  json_serializable: ^6.7.0
```

---

## 运行调试

```bash
flutter run -d windows
flutter run -d macos
flutter run -d linux
```

---

## 构建

```bash
flutter build windows --release
flutter build macos --release
flutter build linux --release
```
