## 核心实现

### 表格组件

```dart
SfDataGrid(
  source: _dataSource,
  columns: [
    GridColumn(columnName: 'pk', label: Text('主键')),
    GridColumn(columnName: 'name', label: Text('字段名')),
    GridColumn(columnName: 'type', label: Text('类型')),
  ],
  allowEditing: true,
)
```

### 校验

```dart
String? validateTableName(String name) {
  if (name.isEmpty) return '表名不能为空';
  if (RegExp(r'[/&:]').hasMatch(name)) return '不能包含特殊字符';
  return null;
}
```
