## 核心实现

### 模板引擎

```dart
import 'package:mustache_template/mustache_template.dart';

class TemplateService {
  String render(String template, Map<String, dynamic> context) {
    return Template(template).renderString(context);
  }
}
```

### DDL 生成

```dart
String generateCreateTableDDL(Entity entity, String dbCode) {
  final template = _getTemplate(dbCode);
  final context = {
    'entity': {
      'title': entity.title,
      'fields': entity.fields.map((f) => {
        'name': f.name,
        'typeDB': getMappedType(f.type, dbCode),
      }),
    },
  };
  return templateService.render(template, context);
}
```

### MySQL 模板

```
CREATE TABLE `{{entity.title}}` (
  {{#entity.fields}}
  `{{name}}` {{typeDB}}{{#pk}} PRIMARY KEY{{/pk}},
  {{/entity.fields}}
);
```
