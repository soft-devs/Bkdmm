# ER 图交互设计规范

## 概述

ER 图画布支持两种交互模式：**预览模式**和**编辑模式**。不同模式下，鼠标事件的响应行为不同。

---

## 模式定义

### 预览模式 (Move Mode)

只读查看模式，用户可以浏览 ER 图，但不能修改。

### 编辑模式 (Edit Mode)

编辑操作模式，用户可以拖动节点、创建关系、框选多个对象。

---

## 事件行为矩阵

### 预览模式

| 事件类型 | 目标对象 | 行为 |
|---------|---------|------|
| 左键按下 | 画布空白区域 | 开始拖动画布 |
| 左键移动 | 画布空白区域 | 平移画布 |
| 左键释放 | 画布空白区域 | 结束拖动画布 |
| 左键按下 | 表节点 | **无响应**（事件穿透到画布） |
| 左键点击 | 表节点 | **无响应** |
| 左键双击 | 表节点 | 打开**预览弹窗**（只读） |
| 右键 | 任意 | **无动作** |
| 滚轮 | 任意 | 缩放画布（以鼠标位置为中心） |

### 编辑模式

| 事件类型 | 目标对象 | 行为 |
|---------|---------|------|
| 左键按下 | 画布空白区域 | 开始**框选操作** |
| 左键移动 | 画布空白区域 | 绘制选区矩形 |
| 左键释放 | 画布空白区域 | 完成框选，选中区域内所有节点 |
| 左键按下 | 表节点 | 开始**拖动节点** |
| 左键移动 | 表节点 | 移动节点位置 |
| 左键释放 | 表节点 | 结束拖动，保存位置 |
| 左键点击 | 表节点 | 选中/取消选中节点 |
| 左键双击 | 表节点 | 打开**编辑弹窗** |
| 左键点击 | 字段锚点 | 开始/完成连线 |
| 右键按下 | 任意 | 开始拖动画布 |
| 右键移动 | 任意 | 平移画布 |
| 右键释放 | 任意 | 结束拖动画布 |
| 滚轮 | 任意 | 缩放画布（以鼠标位置为中心） |

## 视觉反馈

### 光标样式

| 模式 | 区域 | 光标 |
|------|------|------|
| 预览 | 画布 | `grab` |
| 预览 | 拖动画布中 | `grabbing` |
| 预览 | 表节点 | `default`（无特殊指示） |
| 编辑 | 画布 | `crosshair`（框选） |
| 编辑 | 表节点 | `grab` |
| 编辑 | 拖动节点中 | `grabbing` |
| 编辑 | 字段锚点 | `cell`（十字准星） |

### 选中状态

- 单选：点击节点，高亮边框
- 框选：绘制半透明矩形，释放后选中区域内节点
- 多选：Ctrl+点击可追加/移除选中节点

### 连线预览

- 点击锚点后显示虚线预览
- 跟随鼠标移动
- 再次点击目标锚点完成连线
- 点击空白区域或按 Esc 取消

## 关键实现点

### 1. 事件分发机制

```
画布 (Listener)
├── 预览模式：左键拖动 → 平移画布
├── 编辑模式：
│   ├── 左键空白 → 框选
│   ├── 右键 → 平移画布
│   └── 传递给子节点
└── 表节点 (GestureDetector)
    ├── 预览模式：左键无响应，双击 → 预览弹窗
    └── 编辑模式：拖动节点，双击 → 编辑弹窗
```

### 2. 事件拦截策略

**预览模式**：
- 表节点不拦截左键事件，穿透到画布
- 仅双击事件由表节点处理

**编辑模式**：
- 表节点拦截左键事件用于拖动
- 右键事件由画布处理用于平移
- 框选与节点拖动互斥

### 3. 状态管理

```dart
class ERDiagramUIState {
  final ERInteractionMode interactionMode;  // 当前模式
  final Set<String> selectedNodeIds;        // 选中节点
  final String? draggingNodeId;             // 拖动中的节点
  final bool isSelecting;                   // 是否正在框选
  final Rect selectionRect;                 // 框选矩形
  final ERConnectionState connection;       // 连线状态
}
```

## 弹窗类型

### 预览弹窗

- 只读展示实体信息
- 显示字段列表、类型、注释
- 无编辑功能
- 仅一个"关闭"按钮

### 编辑弹窗

- 可编辑所有字段
- 支持添加/删除字段
- 支持修改表名、注释
- 有"保存"和"取消"按钮

---

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Space` + 拖动 | 临时切换到平移模式 |
| `Ctrl + A` | 全选（编辑模式） |
| `Delete` | 删除选中节点（编辑模式） |
| `Esc` | 取消当前操作（连线/框选） |
