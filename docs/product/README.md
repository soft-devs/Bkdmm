# 产品定位

> **阅读时机**: 项目启动时，了解产品目标和核心功能

---

## 产品概述

Bkdmm 是一款**免费、简洁、实用的跨平台数据库模型建模工具**，核心对标 PowerDesigner。

### 与 PowerDesigner 对比

| 维度 | PowerDesigner | Bkdmm |
|------|---------------|-------|
| 价格 | 商业付费 | 完全免费 |
| 复杂度 | 功能全面、配置复杂 | 简化配置、实用至上 |
| 平台 | Windows为主 | Win/Mac/Linux全平台 |
| 上手难度 | 需专业培训 | 5分钟上手 |
| 目标用户 | 专业DBA/架构师 | 开发人员/小团队 |

---

## 目标用户

**核心用户**: 国内金融IT、中小企业开发团队

**用户痛点**:
1. PowerDesigner太贵、太复杂
2. 需要跨平台支持(Mac用户增多)
3. 数据库版本管理困难
4. 代码生成需要手动配置

## 功能模块详解

### 模块(Module)概念
- 模块是数据表的组织单元
- 每个模块包含：数据表(entities) + 关系图(graphCanvas)
- 支持模块级复制/剪切/粘贴

### 数据表(Table)设计
- 字段配置：name、type、chnname、remark、pk、notNull、autoIncrement
- 索引配置：name、fields、type(NORMAL/UNIQUE/FULLTEXT)
- 代码预览：实时显示生成SQL

### 关系图(Relation)可视化
- 拖拽布局节点
- 可视化关系连线
- 支持搜索/缩放/导出图片

### 数据类型系统
定义抽象数据类型，映射到各数据库具体类型：

```
抽象类型(IdOrKey) → MYSQL: VARCHAR(32)
                  → ORACLE: VARCHAR2(32)
                  → JAVA: String
```

## 设计亮点

### 1. "零配置"设计理念
新建数据表自动添加默认字段：
```
REVISION      - 乐观锁
CREATED_BY    - 创建人
CREATED_TIME  - 创建时间
UPDATED_BY    - 更新人
UPDATED_TIME  - 更新时间
```

### 2. 数据升级机制
自动检测老版本项目文件并增量升级

### 3. 多Tab工作区
- 支持同时打开多个数据表/关系图
- Tab超宽自动折叠到溢出面板
- Ctrl+E 快速关闭当前Tab
