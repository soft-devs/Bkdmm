## 二、左侧项目视图详解

```
┌─────────────────────────────────────────┐
│ Project                        [−][⚙][×]│  ← 标题栏 (可折叠、设置、关闭)
├─────────────────────────────────────────┤
│ [模块] [表] [关系] [搜索]               │  ← 视图模式切换
├─────────────────────────────────────────┤
│ 🔍 [搜索表、字段...]                    │  ← 搜索过滤框
├─────────────────────────────────────────┤
│                                         │
│ ▼ 📁 用户模块 (UserModule)              │  ← 模块节点 (展开状态)
│   ├─ 📊 user (用户表)            [🔵]   │  ← 表节点 (蓝色=已修改)
│   │   ├─ 🔑 id (主键)                   │  ← 字段节点
│   │   ├─ 📝 username                    │
│   │   ├─ 📧 email                       │
│   │   └─ ⚙ status                       │
│   ├─ 📊 role (角色表)                   │
│   └─ 🔗 user_role (用户角色关系)        │  ← 关系节点
│                                         │
│ ▶ 📁 订单模块 (OrderModule)      [3]    │  ← 折叠状态 (数字=表数量)
│                                         │
│ ▶ 📁 商品模块 (ProductModule)    [3]    │
│                                         │
│ ─────────────────────────────────────── │
│ ▼ 📁 外部类型                           │  ← 数据类型分组
│   ├─ 📄 PhoneNumber                     │
│   └─ 📄 EmailAddress                    │
│                                         │
├─────────────────────────────────────────┤
│ [➕ 模块] [➕ 表] [🔄 刷新]              │  ← 底部操作栏
└─────────────────────────────────────────┘

快捷键提示:
Alt+1 - 显示/隐藏项目视图
Enter - 打开选中项
Space - 展开/折叠
F2    - 重命名
Delete- 删除
```
