## 核心模型

### Project

```dart
class Project {
  final String id;
  final String name;
  final List<Module> modules;
  final DataTypeDomains dataTypeDomains;
}
```

### Module

```dart
class Module {
  final String id;
  final String name;
  final List<Entity> entities;
  final GraphCanvas graphCanvas;
}
```

### Entity

```dart
class Entity {
  final String id;
  final String title;
  final String chnname;
  final List<Field> fields;
  final List<Index> indexes;
}
```

### Field

```dart
class Field {
  final String name;
  final String type;
  final String chnname;
  final bool pk;
  final bool notNull;
  final bool autoIncrement;
}
```

### GraphCanvas

```dart
class GraphCanvas {
  final List<GraphNode> nodes;
  final List<GraphEdge> edges;
}

class GraphNode {
  final String title;  // 格式: 表名:序号
  final double x, y;
}
```

| Integer | INT | Integer |
| Long | BIGINT | Long |
| DateTime | DATETIME | LocalDateTime |
