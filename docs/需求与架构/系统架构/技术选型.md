# 技术选型

> 阅读时机: 了解技术方案决策

---

## Flutter vs Electron

| 指标 | Electron | Flutter |
|------|----------|---------|
| 内存占用 | 300MB+ | 100-150MB |
| 安装包 | 80-150MB | 20-40MB |
| 启动速度 | 2-5s | 0.5-1s |
| ER图生态 | G6成熟 | **需自研** |

**结论**: 选择 Flutter，性能优势明显

---

## 核心选型

| 用途 | 选择 | 说明 |
|------|------|------|
| 状态管理 | Riverpod | 类型安全 |
| 本地存储 | Hive | 高性能 |
| 表格组件 | Syncfusion | 商业级 |
| 模板引擎 | mustache | Dart原生 |
| 文件选择 | file_picker | 跨平台 |

---

## 图可视化 ⚠️ 重点

Flutter 无成熟 ER 图组件，需用 **CustomPainter 自研**

```
ERDiagramWidget
    ├── InteractiveViewer  # 缩放/平移
    ├── CustomPaint        # 绘制
    └── GestureDetector    # 交互
```

详见: [ER图编辑器](../../开发文档/功能模块/ER图编辑器.md)
