## 分层架构

```
┌─────────────────────────────────────────┐
│              表现层                      │
│  HomeView  WorkspaceView  EntityEditor  │
├─────────────────────────────────────────┤
│              业务逻辑层                  │
│  ProjectService  ModelingService        │
├─────────────────────────────────────────┤
│              数据访问层                  │
│  FileStore  TemplateEngine  GraphEngine │
├─────────────────────────────────────────┤
│              基础设施层                  │
│  Flutter  Riverpod  Hive  Platform APIs │
└─────────────────────────────────────────┘
```

---

## 技术架构

```
┌─────────────────────────────────────────┐
│           Riverpod 状态管理              │
│  ProjectProvider  EditorProvider        │
├─────────────────────────────────────────┤
│              UI 组件层                   │
│  Material  CustomPaint  Syncfusion      │
├─────────────────────────────────────────┤
│              服务层                      │
│  FileService  TemplateSvc  DataTypeSvc  │
├─────────────────────────────────────────┤
│              数据持久层                  │
│  Hive(缓存)  JSON File(项目)  Templates │
└─────────────────────────────────────────┘
```

---

## 数据流

```
用户操作 → Widget → Provider → Service → 持久化 → UI更新
```
