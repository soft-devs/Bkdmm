# 功能规格

> 阅读时机: 详细了解各功能需求

---

## 1. 项目管理

| 功能 | 说明 |
|------|------|
| 创建项目 | 生成 .bkdmm.json 文件 |
| 打开项目 | 文件选择对话框 |
| 项目历史 | 记录最近20个项目 |

---

## 2. 数据表编辑

### 字段属性

| 属性 | 类型 | 说明 |
|------|------|------|
| name | string | 字段名 |
| type | string | 数据类型 |
| chnname | string | 中文名 |
| pk | boolean | 主键 |
| notNull | boolean | 非空 |
| autoIncrement | boolean | 自增 |

### 约束校验

- 表名/字段名不能包含 `/`、`&`、`:`
- 同模块内表名唯一
- 同表内字段名唯一

---

## 3. ER图编辑器 ⚠️

| 功能 | 说明 | 难度 |
|------|------|------|
| 节点显示 | 表名+字段列表 | 中 |
| 拖拽移动 | 拖动节点 | 中 |
| 缩放/平移 | 画布操作 | 低 |
| 连线显示 | 表间关系 | 中 |
| 连线编辑 | 创建/删除 | 高 |
| 导出图片 | PNG/JPG | 中 |
| 自动布局 | 自动排列 | 高 |
